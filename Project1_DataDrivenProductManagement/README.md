# Data-Driven Product Management

A market analysis project exploring global demand for digital fitness classes using Google Trends data.

## Overview

As a product manager for a fitness studio, this project analyzes search trend data to gauge demand and identify growth opportunities for digital fitness products and services.

## Analysis

- **Global workout trends** — worldwide interest in "workout" (2018–2023)
- **Keyword comparison** — home workout vs. gym workout vs. home gym
- **Geographic breakdown** — which countries show the highest fitness interest
