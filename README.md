# DataCamp Real-world Projects

A collection of data analysis projects completed as part of DataCamp's real-world project series.

## Projects

### 1. Data-Driven Product Management
> Fitness studio market analysis using Google Trends data.

Explores demand for digital fitness classes by analyzing keyword search trends globally and by country. Covers:
- Global workout interest over time
- Home workout vs. gym workout trends
- Geographic distribution of fitness interest

**Tools:** Python, Pandas, Matplotlib

**Data:** Google Trends (2018–2023)
