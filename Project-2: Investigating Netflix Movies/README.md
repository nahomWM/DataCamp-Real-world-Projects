# Investigating Netflix Movies

A data analysis project exploring Netflix movie data with a focus on 1990s films.

## Overview

Analyzes the Netflix catalog to understand trends in 1990s movies looking at duration, genre, and the prevalence of short action films.

## Analysis

- **Load & explore** the Netflix dataset
- **Filter 1990s movies** from the full catalog
- **Find the most common movie duration** in the 1990s
- **Count short action movies** (under 90 minutes) from the 1990s
- **Plot duration distribution** of 1990s movies
